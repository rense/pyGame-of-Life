# pyGame of Life
Conway's Game of Life - written in pygame

## Instructions

- Create a virtualenv
- $ pip install pygame
- $ python main.py
- Enjoy!

## Instructions

While paused, click cells to draw. 

- spacebar - toggle pause
- escape - reset game!
- numpad-plus - speed up game
- numpad-minus - speed down game
- q - quit
- w - close/quit
